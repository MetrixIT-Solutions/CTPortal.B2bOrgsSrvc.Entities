# CTPortalPartners.EntitiesApi
CT(Candidates Tracking) Portal Partners - B2B Partners Entities/Organizations API Service.

## Getting started

Clone an existing Git repository with the following command:

1. cd ../workspace/CTPortal/B2B/Services/Orgs/
2. Clone the repo using following git command: `git clone https://github.com/MetrixIT-Solutions/CTPortal.B2bOrgsSrvc.Entities.git`
3. Go to the appropriate directory to start the Api service: `cd CTPortal.B2bOrgsSrvc.Entities`
4. Node version `>=20.16.0` and npm version `>=10.8.1`
5. Run `yarn` or `npm install` to install dependencies
6. Run the command: `npm run start`
7. You can find the sevice status on browser with the url `http://localhost:3404`

Push an existing Git repository with the following command:

```
cd existing_repo
git remote add origin https://github.com/MetrixIT-Solutions/CTPortal.B2bOrgsSrvc.Entities.git
git branch -M main
git push -uf origin main
```
